# auto_report_101
## Скрипт автоматизирует формирование отчётности по банкам на основе открытых данных Центрального Банка РФ (форма 101) и статистики по рынку кредитования (ОБС).
### Как это работает
1) Загрузка данных:
- Скрипт проверяет последний скачанный архив формы 101 в указанной директории (дата определяется по имени файла).
- Пытается загрузить новый архив за следующий отчётный период.
- Если свежие данные недоступны, программа завершается с сообщением об ошибке и кодом sys.exit(1).

2) Распаковка и подготовка:
- Все найденные архивы распаковываются в указанную папку.
- Затем выполняется обработка всех .dbf-файлов.
- Для работы требуется файл-справочник Счета.xlsx, который по умолчанию лежит в корневой папке проекта.
- Промежуточные обработанные данные сохраняются отдельно в формате csv для возможной последующей проверки.

3) Обработка ОБС-данных
- Файл ОБС берётся из той же директории, куда был загружен.

4) Формирование итогового отчёта
- На выходе создаётся сводная таблица в формате .xlsx — финальный результат работы скрипта.

Начиная с этапа обработки .dbf-файлов и до самого конца скрипта выполняется основной вычислительный код.
Эта часть не требует изменений при штатном использовании за исключением редактирования путей к файлам и директориям (место загрузки архивов, расположение справочника Счета.xlsx и папка для сохранения результатов).

### В репозитории приложено:
1) Файл "Авто-отчет.py" - основной скрипт
2) Папка "101_rar" с актуальными файлами dbf формы 101 (до 2026.05.01)
3) Архив "101-20260501.rar" - для проверки сриптом последней даты и скачивания актуального на следующий месяц
4) Файл "Счета.xlsx" (находится в расположении проекта) - для отбора данных по релевантным счетам

### Требуется (в файле "Авто-отчет.py"):
1) Указать пути, куда будут скачиваться архивы 101 (напр., 101-20260601.rar) и данные ОБС (obs_tabl20с.xlsx). По умолчанию установлено save_dir = r"D:\Downloads"
2) Указать папку, куда будут распакованы архивы 101. По умолчанию создается папка "101_rar" в расположении проекта
3) Указать папку, куда будут сохраняться предварительно обработанные данные в форматте csv. Предварительно output_folder = Path(r"D:\Downloads\Исходные данные 101")
4) Указать папку, куда будет сохраняться результат в виде сводной таблицы xlsx. По умолчанию сохраняется в расположении проекта с наименованием 'Сводные с приростами_{new_date}.xlsx'
5) Настроить вывод сообщения try-except при попытке скачивания архива 101
